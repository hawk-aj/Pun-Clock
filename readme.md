<h2>The Pun Clock</h2>

<p align = right>Python, PyQt5</p>

A simple clock in pyqt which updates and displays puns every 10 seconds. Implementation focus being -

- multithreading, 
- qt word-wrapping, 
- window resizing, 
- general design concepts for pyqt.
